### 1. 임의의 두 점 찾기
- 어떤 기준이 되는 값(`target`)이 존재한다.
- 이걸 이용할 **배열은 정렬되어 있어야 한다.**
```python
# len(arr) = n
arr.sort()
start, end = 0, n-1
target = # 기준이 되는 값



while start < end: # 겹칠 수 있다면 등호를 넣는다
	tmp = arr[start] + arr[end]
	
	if tmp == target: # 타겟 값이 성립하거나, 갱신할 수 있는 상황 등
		pass

	elif tmp < target:
		start += 1

	else:
		end -= 1
```
