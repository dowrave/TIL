### 도커 권한 부여
- Root 유저가 아니라면 아래 명령어 전달하자
```ubuntu
sudo groupadd docker (안해도 됨)
sudo usermod -aG docker $USER

# 위까지 입력 후 재로그인 OR 아래 명령어 실행
newgrp docker
```

- 정상적인 경우 permission denied라는 문구가 뜨지 않음
