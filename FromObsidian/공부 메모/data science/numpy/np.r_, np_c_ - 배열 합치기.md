1. `np.r_[]` : 배열을 합침
```python
a = np.array([1, 2])
b = np.array([3, 4])

# 용도 1. 위 아래로 배열을 합침
np.r_[[a], [b]] # [[1, 2], [3, 4]]

# 용도 2. 하나의 벡터로 합침
np.r_[a, b]

# 참고
np.r_[:5] # [0, 1, 2, 3, 4]
np.r_[5:] # [0, 1, 2, 3, 4]
```
- 슬라이싱을 쓸 경우, **시작 범위와 끝 범위 모두를 반드시 명시**해야 함


2. `np.c_[]` :
```python
np.c_[a, b] # [[1, 3], [2, 4]]
np.c_[[a], [b]] # [1, 2, 3, 4]
```
