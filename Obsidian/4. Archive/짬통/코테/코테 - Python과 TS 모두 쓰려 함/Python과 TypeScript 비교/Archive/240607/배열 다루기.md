## 슬라이싱

- 파이썬
```python
arr[a:b]
```
> 인덱스 a부터 b-1까지를 가져옴
> 음의 인덱스 사용 가능

- 타입스크립트
```tsx
arr.slice(a, b)
```
> 인덱스 a부터 b-1까지 슬라이싱함
> b가 없다면 a부터 끝까지 슬라이싱함
> b에는 음의 인덱스가 들어갈 수 있고, 예를 들어 -1이면 뒤에서 가장 마지막 원소를 제외하고 슬라이싱함

## 역순으로 나열하기
- 파이썬
```python
arr[::-1]
```
> 문자열도 바로 가능

- 타입스크립트
	- `.slice()` 함수는 역순을 지원하지 않는다.

- 배열이라면 아래처럼 가능
```tsx
[...arr].reverse();
[...arr].slice().reverse();
```

- **문자열의 경우 `immuatable`이므로, 객체로 바꾼 다음 순서를 뒤집어주고 합쳐야 한다.**
```tsx
str.split('').reverse().join('');
```
