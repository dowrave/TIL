- `SH` : 파형을 바꿈
	- 0% : Sine
	- 25% : Triangle
	- 50% : Saw

- 다음 3개의 슬라이더는 파동의 모양을 조절함
	- `TN` : `Tension`
	- `SK` : `Skew`
	- `SN` : `Sine Shaper`
> 만져보면서 소리가 어떻게 바뀌는지 느껴봐야 함. 파동이 어떻게 변하고 소리가 어떻게 바뀌고는 글로는 설명이 다 안되는 부분임

- `FL` : Low Pass 필터(올릴수록 잘리는 영역이 고음부에서부터 증가)
- `NS` : 화이트노이즈 추가

- 왼쪽의 파형 아래 3개의 아이콘
	- `Half` : 파형을 표시기의 절반만 남김(모든 마이너스 진폭을 제거)
	- `Even` : 파형을 표시기의 왼쪽/오른쪽 절반으로 압축하고, 나머지에는 무음
	- `Absolute(=Mirror)` : 파형을 y축 대칭으로 표시

![[Pasted image 20260312013146.png]]
- `Center` : 파형의 수직 오프셋인 `DC 오프셋`을 제거
	- 정상적인 오디오는 교류 성질(AC)임 : 기준이 0
	- 근데 `Half`를 눌러서 마이너스 부분을 없애면 기준이 `+`로 잡힘. 그런 것들을 처리할 때에 사용함
	- 왜 처리해야 하는가 : 기준이 0이 아니므로 한쪽 끝이 다른쪽보다 먼저 한계치 0dB에 도달해서 소리가 더 빨리 깨짐(헤드룸 이슈)
- `Declick` : 노트가 시작할 때 발생하는 초기 클릭 소리 제거
- `Band Limit` : 오실레이터의 주파수를 20kHZ 이내로 제한, 앨리어싱 방지
	- 효율적이나 소리의 손실이 발생. CPU가 너무 많이 쓰일 경우 사용

- `Pluck` : `Damp` 엔벨로프로 제어됨
![[Pasted image 20260312013629.png]]
- **`Pluck`을 활성화하면, 오른쪽의 `FM` 매트릭스에서 `오퍼레이터`에 해당하는 행이 비활성화**됨. 즉 다른 오퍼레이터로부터 FM을 받을 수 없는 상태가 됨
- 하지만 이거 자체는 여전히 FM 소스로 사용할 수 있음

- `Phase` : 오실레이터의 시작 위상 설정
	- `Global` : 자유롭게 실행되도록 설정
- `Vol`을 설정해서 FM 양을 제어할 수 있음. RM은 제어하지 못함
- `PE` : `Pitch Envelope`
	- 강의 영상에선 소리가 바뀌는데 나는 소리가 안 바뀜;

상단의 `Hz`, `2.0000x`는 오실레이터의 기본 피치를 설정함
1. **오른쪽 상자의 `0`이라면 모든 노트에서 왼쪽 상자에 해당하는 음이 나옴**
2. **오른쪽 상자가 0보다 크다면, 왼쪽 상자는 피치 오프셋으로 작용함**
3. 오른쪽 상자는 피치 배율임 
	- 모든 정수는 고조파 계열`Harmonic Series` 또는 퓨리에 계열`Fourier Series`을 나타냄
		- 퓨리에 계열 : 고유 주파수`Fundamental Frequency`의 배수
	- 여기서는 **옥타브 = 2의 제곱수**라는 것만 알아두면 된다. `2^(n-1)`로 들어감
	- 오른쪽 상자가 1이라면 2의 0제곱인 1, 
	- 2라면 2의 1제곱인 2, 같은 식

![[Pasted image 20260312014646.png]]
- 상단의 `PAN, VOL, MOD, PITCH, PHASE, DAMP, OSC`는 모듈레이션 파라미터
- 하단의 `ENV ~ UNI`는 각 파라미터에 대한 조절기임
- `PAN ~ DAMP`은 `ENV`로 조절할 수 있음
- `OSC`는 나중에 다룸

## 신디사이시스 유형

### Subractive Synthesis(감산 합성)
- 복잡한 시작 사운드에서 주파수를 제거하거나 약화시켜서 소리를 만듦

> 강의가 이해되지 않아서 제미나이에게 물어봄

- `Harmonic Series(배음 열, 여기선 "고조파 계열"로 번역)`
1. 어떤 물체가 특정 주파수로 진동할 때, **자연계에서는 정수배에 해당하는 진동이 동시에 발생**함
2. `특정 주파수`에 해당하는 주파수가 `기본 주파수Fundamental Frequency`가 됨
3. 쌓이는 음들은 일정한 음악적 간격을 가짐

|  순서  | 진동수 비율 |      음악적 간격      | C1 기준 |
| :--: | :----: | :--------------: | :---: |
| 제1배음 |   1배   | 기본음(Fundamental) |  C1   |
| 제2배음 |   2배   |      1옥타브 위      |  C2   |
| 제3배음 |   3배   |     완전 5도 위      |  G2   |
| 제4배음 |   4배   |      2옥타브 위      |  C3   |
| 제5배음 |   5배   |      장 3도 위      |  E3   |
| 제6배음 |   6배   |     완전 5도 위      |  G3   |
> 하모닉 시리즈가 중요한 이유 : 피아노, 기타의 소리를 구분할 수 있는 이유는 하모닉의 구성 비율이 다르기 때문이다. 이를 `음색Timbre`이라고 한다. 어떤 악기는 홀수의 하모닉이 강하고, 어떤 악기는 짝수의 하모닉이 강한 느낌의 차이가 있음.

> 다시 강의로 돌아옴

- 따라서 이러한 `감산 합성` 방식에서는 하모닉이 풍부한 `Saw` 등에서 시작하는 게 좋다.
- **`Saw` 파는 퓨리에 계열의 모든 고조파를 포함**한다. 
> 참고) 퓨리에 시리즈를 음악/소리에서 다룬 걸 하모닉 시리즈라고 함. 같은 얘기임

>[!note]
>- 파형을 이해하는 방법 : 어떤 하모닉(배음)이 들어있는가, 그 배음의 에너지가 얼마나 빨리 줄어드는가
>- `Saw` : 모든 배음이 있음. 배음이 커질수록 `1/n`으로 작아짐
>- `Square` : 홀수 배음만 포함, `1/n`으로 작아짐(1, 1/3, 1/5, ...)
>- `Triangle` : 홀수 배음만 포함, `1/n^2`으로 작아짐(1, 1/9, 1/25, ...)
>- 짝수 배음이 포함될수록 풍성하고 화려해지며(Saw, Pulse) 홀수 배음이 지배적이라면 공허하고 목관의 느낌이 강해짐

...어쨌든, **모든 감산 합성 예제는 가장 복잡한 기본 파형인 `Saw`파에서 출발**한다.

- **감산 혼합 예시 : `Supersaw`, `Acid Bass`, `Reese Bass`**
	- 이런 "소리를 뜻하는 단어"는 알아둘 필요가 있다고 생각함
#### Supersaw 예시

- 설명만 적고, 다른 강의에서 찾아온 방법을 적용해봄

### ★ FM Synthesis(주파수Frequency 변조Modulation 합성)
- 하나의 파형`캐리어Carrier`의 주파수를 다른 파형`모듈레이터Modulator`로 빠르게 변화시켜 복잡한 배음을 만들어내는 방식
	- 파동 간의 직접적인 덧뺄셈보다는, 아래 개념에 가까움. $$Output = \sin(Carrier\_Frequency + Modulator)$$
- `Gemini`의 설명(영상만으로는 이해가 잘 안됨)
- **모듈레이터의 주파수는 일종의 배음 간격**
	- 모듈레이터의 주파수도 `20 ~ 20khZ`로 정해진다. 
- 일반적으로 배음은 그 간격이 정해져 있지만, **FM 합성은 모듈레이터 주파수를 마음대로 조절할 수 있기 때문에 배음 간격을 인위적으로 뒤틀 수 있다**는 것에 의의가 있음
- 이 개념이 바로 `Sytrus`의 `Ratio` 개념
![[Pasted image 20260314011052.png]]
- `x 박스` : 건반을 누를 때 발생하는 기본 주파수에 이 값을 곱한다.
	- 이게 배음 간의 간격을 결정함
	- 정수 비율 : 조화로운 소리. 음악적이고 부드러운 소리가 난다.
	- 소수 비율(1.414, 2.718..) : 비배음적 소리
		- 배음 간격이 비틀리면서 불협화음적 배음이 나타난다.
		- 금속이 부딪히는 소리, 종 소리, 산업용 베이스 사운드 등이 나온다.
		- 악기 중에서는 종, 심벌즈가 이런 배음 구조를 갖는다. 
- `Hz 박스` : `x`로 곱해진 주파수에서 값을 더하거나 뺀다.
	- 일반적으로 0으로 두지만, 음정을 미세하게 비트는 `Detune`으로 소리를 두껍게 만들거나 특수한 효과를 줄 때 사용한다.

- 여기선 주파수 변조라는 표현을 쓰지만, **현대의 FM 신디사이저는 대부분 `위상 변조Phase Modulation, PM`을 사용한다. 위상 변조가 더 안정적이고 음악적으로 유용한 소리를 만든다.**
