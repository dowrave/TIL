## 3x OSC
- [[FL Studio - 3x OSC]]도 참고
- 이름처럼 3개의 오실레이터로 구성되어 있음
![[Pasted image 20260214174919.png]]
우선 2번, 3번의 가장 오른쪽에 있는 노브를 다 왼쪽으로 돌려서 1번만 보면서 실험함


### 파형
- `커스텀` : 오디오파일을 파형으로 가져와서 사용하는 구성(오실레이터에서 톤을 생성하지 않음)
- 파형에 대해선 뒤에서 자세히 다룸
### Phase OFS(페이즈 오프셋)
- **L, R 파동이 시작되는 시간에 살짝의 변화**를 줘서 좌우 스피커에서 동시에 동일한 파동이 나오지 않게 함.
- 즉, **스테레오감을 주는 요소**라고만 이해하고 넘겨도 무방
- 위/아래는 L, R 중에 어떤 방향의 파동을 미루거나 당길 것이냐의 얘기임

### Detune
- L, R 파동의 주파수를 미세하게 비틀어서 다르게 해줌
- 여기서 `맥놀이Beating` 현상이 발생한다. '웅웅' 거리는 진동이 발생함. 
- 즉 
	- `Detune = 0`일 경우 소리가 상대적으로 딱딱하고
	- 살짝만 줘도(`cent 단위`) 웅웅거리는 진동이 느껴짐. 
- 역시 **스테레오감을 주는 요소**지만, 너무 값이 크면 악기 자체의 튜닝이 틀어짐

### Coarse
- 오실레이터의 피치를 세미톤 단위로 조절함.
- (강사는) **화음을 위해 `12` 단위로만 조절**함 `0, 12, 24, 36, 48`

### Fine Tune
- 피치를 미세 조정. `100 Cent`까지 조절 가능하다. `100 Cent = 1 Semitone`

### Panning
- 검은 리본 모양. 

### Level Control
- 2, 3번에만 있는 볼륨 컨트롤. 1번 악기는 항상 100%로 나간다.

### AM OSC 3
- 3번째 오실레이터에 적용되는 함수롤 바꿔서 2번째 오실레이터에도 적용됨

### HQ
- 얼라이싱을 줄여서 음질을 늘려줌

### Phase RAND
- 오실레이터의 `Phase`에 랜덤을 가함
- 살짝만 주면 자연스러운 효과가 나고, 올리면 스테레오 효과가 증가.

### Sine
- 가장 부드러움. 
- EQ로 찍어보면 **1개의 주파수만 피크**로 하는 걸 볼 수 있다.
- 여기서 **기준이 되는 음(=찍은 음)을 `Fundamental`** 이라고 정의
![[Pasted image 20260214182100.png]]
> 찍는 순간. 시간이 지나면 언덕 부분은 다 내려감



### Triangle
- `Sine`에 비해서 훨씬 풍성함 : 그 자체로 `배음Harmonics`을 이룸
- **여러 개의 피크가 있더라도 `Fundamental`의 피크가 가장 높음**
![[Pasted image 20260214182243.png]]


### Square
- `Triangle`보다도 훨씬 배음이 많이 들어갔음
- 풍부한 텍스쳐를 얻을 수 있음`much richer than both sine and triangle waves`
 ![[Pasted image 20260214182547.png]]

- `Triangle`과 비교하면,
	- `Triangle`은 피크음이 뚜렷하게 들리지만 피크를 벗어난 다른 음들이 묻히는 느낌이 있음
	- `Square`는 그 음 주위의 음들이 확실하게 잘 들리는 느낌

### Sawtooth
- `Square`보다도 풍성한 텍스쳐를 갖는다.
- 홀수 배음과 짝수 배음을 모두 갖는다. 
	- `Triangle, Square`는 홀수 배음만을 가짐
![[Pasted image 20260214183129.png]]
> - 그래프는 `Square`와 비슷해보이지만, **가장 낮은 음을 기준으로 보면 추가적인 피크들이 나타난 게 보일 거임**
> - 그리고 (체감이지만) 이쪽이 더 전자음악의 느낌이 강함

**여기까지가 모든 오실레이터에서 사용하는 기본적인 파형들임**

## 3x OSC 사용해보기
- 3번째 오실레이터는 끄고, 1번과 2번의 설정을 동일하게 맞춰줌
- 실습은 따라해보면서 체험하면 된다. 쓸 내용이 없으면 그냥 과정만 정리해둠


1. 2번에만 **`Fine Tuning`** 을 살짝 오른쪽으로 돌리기 
	- 꾹 누르고 있어도 스스로 커졌다가 작아졌다가 하는 움직임이 느껴짐

2. 둘 모두 `Sine` -> `Square`로 변경 

3. 2번만 **`Detune`** 살짝 올려줌
	- 일종의 코러스 효과가 남 (위에서 스테레오 효과라고 하기도 했음)
	- 이건 사운드 밸런스에서 필수적인 요소는 아니라고 함

4. 파형에 있는 `Invert`아이콘 클릭
	- 다른 모든 조건이 동일하기 때문에 말 그대로 위상만 뒤집은 상태
	- 그러면 아무 것도 안 들린다.
	- 여러 파동을 섞으면서 소리를 만들 때 유용한 테크닉임

4번 예시
![[Pasted image 20260214184713.png]]
> `Square + Saw`


![[Pasted image 20260214184646.png]]
> `Square + Saw(Inverted)`

**어느쪽이 뚜렷하게 들리는가?** : **아래 버전**

5. 이 상태에서 1번을 Saw로 바꾸면? : 당연히 아무 소리도 들리지 않음
	- 근데 그 상태에서 2번을 Fine Tuning 하면? : 바로 소리가 남(조금만 돌아가도 남)
	- 그래서 **완벽하게 위상이 차단되는 건 똑같은 파동이 위상이 뒤집혔을 떄 뿐**임(피치만 살짝 달라졌는데도 바로 소리가 나니까)

## Simsynth
![[Pasted image 20260214185119.png]]
- 80년대 신스를 바탕으로 만들었음. 깔끔하고 만지기 쉽다. 
- `Polyphonic`이라서 한 번에 2개 이상의 노트를 연주할 수도 있음
	- `Polyphonic` : `다성 음악`이라는 뜻으로 독립적인 멜로디가 둘 이상의 라인에 동시에 구성되는 어우러짐의 일종.
		- TMI) 블루 아카이브 최종장에 나오는 BGM 제목이기도 하다.

- 사전 세팅
	- 우측 상단 - `Default` 프리셋 설정
	- 필터 Off : `SVF`의 오른쪽에 있는 빨간 점 클릭해서 끔
	- 코러스 Off : 우측 하단의 주황색 점 클릭
	- OSC 2, 3번 Off

![[Pasted image 20260214185550.png]]
- 3x OSC와 공통되는 요소들
	- 상단 5개의 파형
	- 음 높이 조절
		- `CRS(Coarse)` : `+- 24 semitones` 
		- `FINE(Fine Tuning)` : `+- 100 cents`
	- 볼륨 조절 
		- `Level` : 3x OSC의 `Level Controller`와는 다르게 **각 오실레이터의 볼륨을 설정**함

- 3x OSC와 차이점이 있는 요소들
	- **`WARM`** : 기본 파형을 `Detune`한 파형을 복사해서 소리를 부드럽게 만듦
		- `Detune` 여부를 보려면 소리가 계속 나고 있을 때 `EQ`에서 파동이 요동치는 정도가 증가하는지를 보면 된다.
	- **`1, 2`** : 각각 1, 2 옥타브가 올라간 파동을 함께 재생할지 여부
	- **`Ring 1x2`** : 2번 오실레이터가 켜졌을 때만 동작. `Ring Modulation`을 가한다.
		- `Ring Modulation = Aplitude Modulation`이라고도 함
	- **`PW(Pulse Width)`** 
		- **`Square, Sine`일 때만 동작**함.(불이 연두색으로 들어옴)
		- 이거는 돌려보면서 느낄 수 밖에 없을 듯? **돌리는 양에 따라 톤이 달라짐**
	- 오실레이터의 피치 조절하는 노드들(`LFO`, `ENV`)은 아래에서 다룸.

### Filters
- **필터 : 신디사이저 내부의 소리를 만질 수 있게 해주는 간단한 EQ**

- `SimSynth`는 `SVF`라는 이름으로 들어가 있다.

> 여기서 `Env`만 기본 설정으로 돌리고 오실레이터 3개 다 켠 채로 테스트

- `SVF Off`
![[Pasted image 20260214191138.png]]

- `SVF On`
![[Pasted image 20260214191159.png]]
> 고주파의 경우 처음 소리가 나올 때에만 한 번 치솟은 다음 내려감

**고주파가 잘려나간 이유는 `Low Pass Filter`** 가 걸려 있기 때문. 

- **`Cutoff(CUT)`** 
	- 왼쪽으로 돌릴수록 높은 주파수를 많이 쳐냄. 
	- 오른쪽으로 돌릴수록 높은 주파수를 남김.

- **`Emphasis(EMPH)` : `Resonance`을 조절**
	- 대충 울리는 효과와 관련된 노드. 
	- 테스트 기준으로 MID 영역이 살짝 올라오고 진폭이 올라가는 빈도도 상승하는 느낌.
	- `Cutoff` 없이 쓸 경우 고주파의 진동이 눈에 띄기 때문에 이 둘을 조합해서 쓰는 듯 함. 

- `High` : 0이면 `low pass filter`로, 100이면 `high pass filter`로 동작
	- 필터의 동작
		- 대부분의 신디사이저에선 필터를 고를 수 있음
		- **`low pass filter` : 저주파를 통과, 고주파를 깎음**
			- 뒤에 붙는 숫자`6, 12, 24`는 소리를 얼마나 깎아내는지를 의미한다. 
				- `db/Octave`를 의미함.
			- 숫자가 클수록 칼같이 잘라내고 딱딱한 느낌을 줌.
		- **`high pass filter` : 고주파를 통과, 저주파를 깎음**
		- 이 둘은 `EQ`에서 직접 깎는 것과 똑같은 효과임

> 일단 `Cut`이랑 의미가 좀 헷갈리는데
> - `Cut`은 기본 50%값이라 `SVF`가 켜지면 고주파를 어느 정도 깎으면서 시작함. **오른쪽 100%라면 모든 소리가 통과되고, 왼쪽 0%로 가면 고주파가 많이 깎임** 
> 	- 그래서 얘를 일종의 Low Pass Filter 역할을 한다고 생각해도 무방함
> - `High`는 오른쪽으로 돌릴수록 저주파가 많이 깎이는데, 한편으로는 고주파가 다시 살아나기도 함

![[Pasted image 20260215001124.png]]
> 저 붉은 부분이 `High`를 어느 정도 걸어줬을 때 살아나는 값.  더 키우면 저 골 같은 부분이 왼쪽으로 밀리고, 왼쪽의 진폭은 줄어들면서 오른쪽의 진폭은 커진다. 

- 저 골 부분을 `notch`라고 한다. 
- 이런 식으로 `High`를 고정하고 `Cut`을 만져서 필터를 조절할 순 있음
- 근데 강사님도 지금 같은 `high, low`를 조합한 `notch filter`를 실제로 많이 쓴 적은 없다고 함

- 그래서 `High` 노드에 대해선 아래처럼 정리하고 넘김
	- **왼쪽 만땅 : `low pass filter`**
	- **오른쪽 만땅 : `high pass filter`**
	- **이 둘은 `cut`이랑은 좀 다른 듯**

- `Band Pass Filter`
	- `low, high`가 동시에 적용되는 개념

- `KB`
	- 기본적으로 `Cut`을 걸면 고음부가 답답하게 들리는데 `KB`를 걸면 고음부가 조금 살아나는 느낌은 있음 
	- 음 높이에 따른 `Cut-Off` 영역이 조금씩 달라지는 효과

## ★엔벨로프(Envelope) - Amplitude
> Simsynth 우측 상단
![[Pasted image 20260215002153.png]]
- `Amplitude Envelope`에 대해서만 다룬다.
- **시간에 따른 볼륨을 컨트롤**함.

- `LEVEL` : 전체 아웃풋 볼륨값

- 엔벨로프의 표준은 `ADSR`이다. 
- **`Attack` : 침묵 -> 최고 볼륨까지의 시간**
	- `Attack`이 증가하면 볼륨이 서서히 증가하므로 일종의 페이드인 같은 효과가 나지만, 너무 길면 소리가 나는지도 모를 수도 있음
- **`Decay` : 최고 볼륨 -> 서스테인 레벨까지 내려가는 데 걸리는 시간**
- **`Sustain` : 노트를 누르고 있는 동안의 볼륨**
	- **`ADSR` 중에서 얘만 볼륨값이고 나머지는 시간**임
	- `Sustain` 값이 최대라면 `Decay`는 의미가 없어짐
- **`Release` : 노트가 더 이상 연주되지 않을 때 소리가 완전히 사라지기까지 걸리는 시간**

- `AMP Envelope`를 쓰고 싶다면, 어떤 소리를 만들고 싶을지 생각해야 한다.
	- `Attack` : 누르자마자 소리가 나오는 것? 아니면 페이드 인?
	- 노트를 오래 울리게 하고 싶다면 `long decay & high sustain`
	- 클릭감 / 퍼커션 같은 느낌을 내고 싶다면 `short attack & short decay & low sustain`
	- 노트를 천천히 사라지게 하고 싶다면 `medium sustain & long release`

- 예시 
	- 베이스 만들기 : `No Attack, Full Sustain, No Release`
		- 테스트해보니까 `Sine`이 잘 어울리는 듯. 나머지는 좀 색깔이 들어간 느낌.
	- `Pad` 사운드 만들기 : `long Attack, long Release`
	- `Pluck` 사운드 만들기 : `short Attack, Medium Decay, low Sustain`

- 왼쪽의 OSC에는 `ENV` 노드가 있음
![[Pasted image 20260215014711.png]]
- **`AMP` 엔벨로프를 이용해서 피치를 조절할 수 있는 기능**임
- 볼륨이 높아질 때 
	- 왼쪽으로 돌리면 피치가 낮아지고
	- 오른쪽으로 돌리면 피치가 높아짐

- 이전엔 `Amplitude Envelope`을 봤다면 이번엔 `Filter Envelope`을 볼 차례

![[Pasted image 20260215010158.png]]
> 아래 4개는 `Filter` 에서 이미 다뤘음

- **`ENV(Envelope Amount Dial)` : 필터 인벨로프에서 소리를 가장 많이 바꾸는 요소**
	- 기본값인 가운데에 오면 소리에 아무 영향도 주지 않음

강의를 들어도 뭔 말인지 모르겠으니 제미나이에게 물어보며 정리했다.

>[!note]
>1. **필터 엔벨로프는 노트가 입력되고 나서 시간에 따라 `Cutoff` 다이얼을 돌리는 역할**을 함. 즉, **노트가 연주되는 동안의 필터링 기준점에 대한 엔벨로프**이다.
>2. 기본 : 로우 패스 필터 기준으로 `CUT`에 설정된 지점 이상의 소리에 필터링이 걸린다.
>3. "엔벨로프 값이 `0`일 때 가장 먹먹하다" : 필터링이 가장 많이 적용되고 있기 때문(즉 `Cut`이라는 가장 낮은 주파수 이상의 소리에 다 필터링이 걸리고 있기 때문)
>4. 필터 엔벨로프가 적용되면, **`CUT`에서 `ENV`까지 기준치가 변화할 수 있음**
>5. **`Attack` : 기준점이 최대치까지 올라가는 데에 걸리는 시간.** 
>	- 즉 `Attack` 시간 시점에 필터링이 가장 적게 걸리므로 소리도 가장 밝게 들림
>6. **`Decay` : `Attack`에서 `Sustain`까지 기준점이 내려가는 데에 걸리는 시간**
>7. **`Sustain` : 노트가 유지되는 동안 `Cutoff` 다이얼의 위치를 결정함**
>	- 얘만 '시간'이 아니라 '레벨'이라는 것도 기억해두자.
>8. **`Release` : 연주되지 않을 때 원래 `Cutoff`로 돌아오는 데 걸리는 시간**

- 두 엔벨로프를 비교해서 정리하면 아래와 같다.

| **구분**      | **Amplitude Envelope (Amp Env)** | **Filter Envelope (Filter Env)** |
| ----------- | -------------------------------- | -------------------------------- |
| **조절 대상**   | **음량 (Volume / Gain)**           | **음색 (Cutoff Frequency)**        |
| **핵심 질문**   | "소리가 언제 커지고 언제 작아지는가?"           | "소리가 언제 밝아지고 언제 어두워지는가?"         |
| **0의 의미**   | 완전한 침묵 (소리가 안 남)                 | 설정한 Cutoff 값 (가장 먹먹한 상태)         |
| **Attack**  | 소리가 들리기 시작해 최대 음량까지 가는 시간        | 소리가 열리기 시작해 가장 밝아지는 시간           |
| **Sustain** | 건반을 누르고 있을 때 유지되는 **음량**         | 건반을 누르고 있을 때 유지되는 **밝기**         |
| **Release** | 건반을 뗀 후 소리가 **완전히 사라지는** 시간      | 건반을 뗀 후 소리가 **원래 설정값으로 닫히는** 시간  |
| **필수 여부**   | 필수 (없으면 소리가 안 나거나 안 멈춤)          | 선택 (소리의 움직임을 줄 때 사용)             |

- `Attack, Decay = 0`이라면 필터 엔벨로프는 적용되지 않음
- `Filter Envelope`의 `Release`는 `Amp Envelope`의 `Release`가 충분히 길지 않으면 체감하기 어려움
	- `Amp`의 `Release` > `Filter`의 `Release`여야 효과가 나옴.

	- 즉 컷오프가 저주파로 내려갔다가 다시 올라가는 동작

### Track Amp
- 활성화하면 `Filter Envelope` 대신 `Amplitude Envelope`의 그래프를 따라감
- 이걸 쓰더라도 `Env` 값 자체는 미리 만져둬야 함. 0이면 필터 엔벨로프는 동작 안 함.

## LFO(Low Frequency Osciliator)

![[Pasted image 20260215015152.png]]
- 들을 수 없는 수준의 저주파를 생성함

- 주의 : `LFO` 효과가 적용되려면 `SVF`에 있는 `LFO` 게이지가 중앙에서 벗어나야 함

- 오른쪽으로 **서서히 증가시키면 고주파 부분이 서서히 요동치는 걸 볼 수 있고, 왼쪽으로 돌리면 저주파가 서서히 요동치는 걸 볼 수 있음**(대신 기존 음이 없어짐)

- LFO의 파형
	- 사각형 : 0-1이 반복되는 느낌의 효과. 최대-최솟값만 반복된다.
	- 톱니 : 삐용삐용하는 느낌의 효과. 
	- 삼각형 :  서서히 증가하고 감소하는, 부드러운 효과
	- 노이즈 : 랜덤
	- 다른 신스에서는 사인파가 있는 경우도 있음

- `RATE` : LFO의 속도 조절. 빈도라고 하는 게 더 맞나? 
- `Retrigger`
	- 사전 세팅 : SVF, OSC2, 3번 다 끄고 1번의 LFO를 만땅으로 돌림
	- 이 상태로 그냥 돌리면 `OSC 삼각형, LFO 삼각형 기준`으로 상승 - 하강을 반복하는 느낌이 남
	- 노트 하나만 반복적으로 클릭하는 상황 
		- `Retrigger` 비활성화 
			- 누를 때마다 다르게 들림 
			- 이거는 **`LFO` 자체가 항상 순환하는 구조**이기 때문에 그렇다. 
		- `Retrigger` 활성화
			- 항상 똑같이 들림
			- 노트를 누를 때마다 LFO가 다시 실행`Re-Trigger`되기 때문이다. 
	- 그런데 노트를 하나씩 순차적으로 입력해서 총 3개의 노트를 입력한다면
		- `Retrigger` 비활성화 : 3개의 노트의 상승 / 하강이 동시에 진행됨
		- `Retrigger` 활성화 : 3개의 노트의 상승 / 하강이 각각 진행됨
			- 이는 **`LFO`가 각 노트에 대해 재실행되기 때문**이다. 
	- 결론) 
		- **코드를 구성하는 악기에 대해서는 `Retrigger`를 비활성화**해야 함
		- 반대로 동시에 음이 연주될 필요가 없다면 괜찮겠지요
- `Delay`
	- `Retrigger` 활성화 시에만 활성화됨
	- 노트가 연주될 때, LFO의 강도를 **서서히** 상승시킴
	- LFO가 최대 강도에 도달하는 데까지 걸리는 시간을 늘리는 개념

- LFO도 마찬가지로 더 업그레이드된 신디사이저에서는 다양한 파라미터에 적용 가능함. 
- 여기서는 필터랑 피치 정도에만 사용할 수 있음
