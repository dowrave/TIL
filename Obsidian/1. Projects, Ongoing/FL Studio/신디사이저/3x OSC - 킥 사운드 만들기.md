즉 우리가 듣는 드럼의 킥 사운드는 **누른 순간 피치가 치솟은 다음 떨어지면서 난다**는 거임
- 하단의 **`Tension` 노브 중 `Decay`에 관한 노브는 `Hold`의 끝 점 ~ `Sustain`으로 떨어질 때 곡선이 어떻게 휘는가를 결정**한다. 
	- 볼륨 / 피치 모두 이거에 의해서도 소리가 많이 차이남.
	- 킥감은 피치가 빨리 떨어지는 게 좋으므로 위처럼 설정

- `Filter`
	- **`MOD X`는 필터 설정.**
		- `Fast LP` : 가볍고 계산이 빠르다. CPU 점유율이 낮지만 거칠고 아날로그 느낌
		- `LP` : 로우 패스 필터
		- `HP` : 하이 패스 필터
		- `BP` : 밴드 패스 필터 - 가운데만 남기고 양 끝을 쳐냄
			- 전화기 목소리, 라디오 등에 사용
		- `BS(Band Stop)` : 대역 차단 필터 - 가운데를 없애고 양 끝을 남김
			- 휘파람, 독특한 위상 변화에 사용
		- `LPx2` : `x2`는 기울기 강화 - 더 칼같이 깎는다고 생각하면 됨
		- `SVF(State Variable Filter)`
			- 상태 변수 필터.
			- 일반 필터보다 소리가 더 아날로그적이고 음악적으로 들린다. 공명값을 올렸을 때 소리가 예쁘게 강조되는 느낌이 있음.
	- **`MOD Y`는 `Reasonance`에 관한 설정.** LP 기준 높일수록 고음부분이 공명에 의해 살아남
		- 좀 특기할 점은 Mod X로 고주파를 깎더라도 **공명하는 부분이 똑같이 왼쪽으로 밀림**
		- 공명이 살아있으면 드럼이 조금 더 딱딱하게 느껴진다. 참고.

> 드럼의 킥 사운드 같은 경우 `LP`와 `Mod X, Y`를 조합하는 것의 의미는 공명이 어디서 울리게 할 것인가 정도에만 의미가 있어 보임. `Mod X`의 경우, 어차피 저음을 쓰니까 고음을 잘라도 큰 의미는 없지만, 공명의 위치를 조절할 수 있기 떄문임

- 기준음 잡기
	- ~~`Channel Rack`은 항상 `C5`를 연주함 ~~
	- `스패너 모양 아이콘Miscellanous Functions`에서 하단의 키보드 건반을 우클릭하면 기준음을 옮길 수 있다. 이렇게 **기준음을 옮긴 다음 C5를 클릭했을 때 원하는 소리**가 나면 설정 완료.
![[Pasted image 20260215235639.png]]

> 그래서 일단 C5를 기준으로 잡긴 했는데, `Piano Roll`에 찍을 때랑 `Channel Rack`에 찍을 때랑 묘하게 동작이 다르다. `Channel Rack`에 찍으면 굉장히 짧은 노트로 찍히는데, 저음에서 웅웅거리는 게 계속 느껴짐. 반면에 `Piano Roll`에 살짝 길이가 있는 노트로 찍으면 그런 현상은 없음.

> 이 문서도 사실 상단의 `Envelope` 2개 설정법만 알면 거의 끝나는 거고 나머지는 짜투리임
