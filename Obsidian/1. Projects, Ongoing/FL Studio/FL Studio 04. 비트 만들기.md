#FLStudio 

## 드럼 패턴

- 악기 소리 찾기 : `Packs/Drums/`
- 강의는 `808 kick`을 사용한다.
	- **디폴트로 설정된 `808 kick`이랑 다른 소리**가 난다. 뭐냐..

**킥 -> 스네어 -> 하이햇 -> 클랩 순서**로 패턴을 짜며,
각각 **악기 찾기 -> 패턴 들으면서 채널랙에 찍는다.**

> 하이햇의 경우 CH가 있고 OH가 있는데
> CH는 Closed Hat으로 짧은 소리
> OH는 Opened Hat으로 긴 소리가 난다.
> 강의는 808 CH로 진행.

만약 규칙적인 패턴을 찍는다면 일일이 찍지 말고 `채널랙의 악기 우클릭 - Fill Each n Step` 기능이 있음

- 스네어와 클랩은 같은 박자로 짜는 경우가 매우 흔하다. 효과가 커지기 때문이다. 

- 채널랙에서 볼륨, 패닝 설정할 수 있다. 이전에도 다뤘으니 패스.
	- 강의에선 `power clap 16`을 쓰는데, 소리가 커서 스네어도 묻힐 정도라서 설정해줌.

>[!note]
>- 비트를 짤 때 강의에서 하는 것처럼 꼭 드럼부터 짜지 않아도 된다. 화음 파트나 베이스부터 먼저 짜도 무방함.
>- 드럼부터 짜는게 빠른 편이라고 추천한다는 언급은 함. 하지만 다른 방법도 틀린 게 아니다.

## Keys 패턴
- `Misc - FL Keys`를 사용한다.
- `Keys`라는 별도의 패턴을 만들고, `Bass`의 작업 내용을 복붙해서 배치해둔다
	- `Bass`의 소리를 듣고 싶은 게 아니라, **베이스에서 작업한 내용을 시각적으로 참고하기 위함임**
	- 그래서 복붙은 해두되 뮤트 처리하고 `Ctrl + 아래` * 3 으로 바닥에 따로 둠

- 노트 선택된 상태에서의 조작법
	- `Ctrl + 위/아래` : 옥타브 단위 이동
	- `Shift + 위/아래` : 키 단위 이동

![[Pasted image 20260108000144.png]]

그래서 `Keys`로는 이런 패턴을 짰는데, 위의 베이스 파트와 비교해보면(옥타브만 내렸을 뿐 C#은 그대로 C#임)

- 바닥의 기준음 기준으로
- 첫 2마디 : C#으로 동일
- 3마디 : A로 동일
- 4마디 : G로 동일 (F#도 있는데, 그래서 4마디에는 F#4도 포함되어 있음)

- 실제로 화음은
	- 첫 2마디 : C# 마이너
	- 3마디 : A 메이저
	- 4마디 : G 메이저 7
		- 7로 만든 이유는 베이스에 F#이 있기 때문

- 패턴을 이렇게 짠 후, 악기 자체를 만져줌
![[Pasted image 20260108000751.png]]
- `Grand Piano -> Rhodes`으로 변경
- `Panning`을 오른쪽으로 살짝 돌려줌 : 1시
- `Treble`을 오른쪽으로 돌림 : 2시
- `Decay` 살짝 증가 : 10시 - 소리의 울림(혹은 유지되는 정도)을 줄여줌
- `Overdrive` 증가 : 12시
	- 소리의 입자감이 거칠어진다(바삭해진다, 서스테인이 길어져서 유지되는 느낌이 증가한다, 배음이 풍성해진다)
	- 컴프레션 효과가 있어서 에너지가 큰 저음이 약해짐
	- 오버드라이브는 중음역대를 부각시킨다 - 날카로운 고음, 쏘는 듯한 중음이 강해지므로 저음이 상대적으로 죽는 효과가 있음

### Arpeggiator
- 진입하기 전에 [[FL Studio - Arpeggio]]도 참고. 
- 위의 `FL Keys - 톱니바퀴 -> 스패너 아이콘 클릭`
![[Pasted image 20260108001311.png]]
- 가운데의 `Arpeggiator`에 주목하자.
- 상단의 아이콘 6개(X부터 ?까지)
	- `위` : 아래에서 위로 음들을 쪼개서 연주
	- `아래` : 위에서 아래로 음들을 쪼개서 연주
	- `위 + 아래` : 아래 -> 위 -> 아래 순으로 반복해서 쪼개서 연주
	- `위 + 아래 + 막대` : 위 + 아래와 동일하되, 가장 낮은 음, 가장 높은 음을 한 번 더 연주
	- `물음표` : 말 그대로 랜덤
	- 여기선 `위 + 아래`로 진행
- `Time`
	- 왼쪽으로 돌릴수록 잘게 쪼개고, 오른쪽으로 돌릴수록 덜 쪼갠다.
	- 좌측 상단의 값을 보면서 수정한다. 여기선 `Arpeggio Rate : 2.00`
- `Gate`
	- 쪼개진 노트가 연주되는 시간, 즉 짧을수록 쪼개진 노트의 연주 시간이 짧아지고 길수록 길어진다.
	- 여기선 기본 설정 유지.

### 다른 악기로 테스트해보기
- 채널랙에서 `Replace - 다른 악기 선택`
- 여기선 `FLEX`을 사용한다.

- 기존 패턴에서 악기만 `FLEX`로 바꿨을 때의 문제점으로
	1. 화음을 연주하지 않고 단일 음만 연주함`Monophonic`
	2. `Arpeggiator`도 적용되지 않음

- 대신 `Arpeggiator`을 적용하면 1번의 문제도 함께 해결된다
- (설정을 복붙하는 줄 알았지만) 이전 Arpeggiator의 세팅이 간단했기에 **똑같이 톱니바퀴 - 스패너**를 열어서 설정해줌

- 기본 악기인 `Dystopian Lead`가 Arpeggiator와 어울리지 않아서, 여러 악기를 들어보고 Chariot Horn을 선택. 

![[Pasted image 20260108002925.png]]
> 프리셋의 소리도 좋지만, 더 좋은 소리를 찾기 위해 악기의 세부 설정도 만질 것을 추천
> `FLEX`의 경우 `MACROS`의 옵션들이 대표적이다. 
> - **신스 뉴비라면 Macros의 옵션부터 만지고(변화가 명료하기 때문에)** 
> - 이해가 더 깊어지면 아래의 세부 필터를 만지는 것을 추천

![[Pasted image 20260111222600.png]]

## 믹싱
- 마스터 레벨에서의 볼륨에 관해 : `0db`보다 아래로 오게 한 상태로 작업하면 ㅇㅋ
	- 더 조용하면 볼륨을 올리면 그만이지만
	- 더 시끄러우면 트랙을 망가뜨릴 수 있음

- 각 **악기의 볼륨을 만지는 것부터 시작**한다.
	- `Keys`의 볼륨이 너무 커서 `-6db`
	- `kick`은 살짝 키움

- **스네어에는 리버브를 추가한다. `fruity reeverb 2`**
	- 스네어에 **흔히 쓰는 효과**임.
	- 알다시피 리버브는 울리는 효과다. 스네어 효과가 너무 크다면 **`wet`을 줄여줄 것.**
	- 여전히 기억할 것) 플러그인을 `bypass`(껐다가 켰다가) 하면서 플러그인의 효과를 체크해볼 것
	- 리버브는 소리를 키워주기 때문에 볼륨을 살짝 죽여서 완충시켜줌

- `Panning`과 `Stereo Separation`
	- `Panning`은 소리가 나는 위치를 조절함
	- `Stereo Separation`
		- 왼쪽으로 돌리면(하늘색) `Wider`
		- 오른쪽으로 돌리면(보라색) `Central`
	- 이번 파트에선 만진 부분 없음
