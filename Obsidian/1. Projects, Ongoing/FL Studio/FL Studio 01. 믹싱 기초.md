#FLStudio 

- 일단 플레이리스트에 만든 패턴을 올릴 때, **각 트랙에는 개별 악기로 구성하는 걸 추천**함
	- 즉 한 개의 트랙에 2개 이상의 악기를 섞는 걸 권장하지 않는다는 뜻임

곡을 완성했다면 믹싱 작업에 들어가면 된다.

1. **볼륨**
	- 기본적으로 어떤 소리가 튀는 걸 원하고 원하지 않는지에 따라 볼륨을 맞추는 것으로 시작함

2. **패닝**
	- 스테레오감을 조성하기 위한 작업.
	- 강의 예제에서는 크게 드럼 / 베이스 / 코드 / 멜로디1 / 멜로디2 / 패드 로 구성되어 있고, 바닥에 화음을 쌓는 역할을 **코드와 패드**가 구성하고 있다.
	- 이 2개를 각각 왼쪽 패닝, 오른쪽 패닝으로 돌림(각각 절반 정도)

3. 플러그인 추가 & EQ 조절
![[Pasted image 20251217020219.png]]
- 플러그인 추가 : `Slot`에 리버브`Fruity Reeverb 2`등을 지정할 수 있음. 종류는 다양함.
- EQ 조절 : 하단 3개의 게이지나 그래프 등을 조작해서 주파수에 따라 소리를 조절하는 기능은 곧바로 사용할 수 있다.
	- 배경음으로 깔아야 하는데 너무 소리가 튄다거나 하는 부분들을 세부적으로 잡아나가면 됨

> 	일반적으로 높이면 찌르는 느낌, 낮추면 먹먹한 느낌이 남

## 악기 묶어서 중간 믹서로 잇기
![[Pasted image 20251221010025.png]]
- 1~5번 채널이 드럼에 관한 믹서라고 할 때, 킥 / 클랩 / 하이햇 / 스네어 등은 채널랙 상에서는 개별 믹서로 들어가게 되고, 디폴트 설정으로 이들은 마스터로 바로 직행한다

- 하지만 **별도의 중간 믹서를 둬서 믹서 -> 믹서로 신호가 가게 처리할 수 있다.**

1. 위에서 1 ~ 5번의 악기 이름 부분들을 ctrl + 드래그로 묶음

![[Pasted image 20251221010251.png]]

2. 6번 채널의 아래에 있는 화살표를 클릭 - `Route to this track only`
	- 묶인 상태에서 바로 클릭해도 되긴 하는 듯

> 여기까지 진행하면 1 ~ 5번 믹서가 6번 믹서로 연결된 상태가 된다.
> 6번 믹서만 설정을 만져줘도 1~5번의 믹서들을 한꺼번에 조절할 수 있음

(물론 6번 믹서는 마스터로 연결되어 있어야 함)

## 구분자 넣기
믹서 우클릭 - `Separator`을 넣으면 왼쪽 믹서와 거리를 두는 구분자가 생김
![[Pasted image 20251221010813.png]]
> 6, 7번에 `Separator`를 넣은 형태


- 믹서에서
- 채널들은 악기별로 구분해서 하나의 그룹을 만들고, 그 하나의 그룹을 출력으로 곧바로 이어준다. 
	- 이 때 악기별로 색을 공통되게 하고 출력 채널은 별도의 색을 갖게 하는 것도 좋은 방법.

- 우선 드럼부터 시작
	- 다른 채널들의 믹서를 끄고
	- 드럼의 구성원들도 볼륨 최하 상태에서 시작함
	- 킥의 볼륨을 맞춘 다음, 스네어, 하이햇, 톰 순서대로 하나씩 맞춰가면 됨
	- 강의에서는 `Shaker`까지 드럼 파트에 포함해서 체크

드럼을 끝냈다면 다른 악기들도 모두 켜서 잘 녹아드는지 체크


### 패닝 관련
> 1. **킥, 스네어를 항상 중심에 놓고 작업할 것.** 별도의 이유가 없다면 이 둘이 항상 곡의 중심이 된다.
> 2. **높은 소리**, 즉 하이햇이나 쉐이커 같은 소리는 각각 **좌우 패닝을 줘서** 스테레오감을 형성한다. 
> 	- 비슷한 원리로 `Tom`도 높은 톰과 낮은 톰을 각각 좌우 패닝을 줌
> 3. 베이스나 `Lead(메인 멜로디)` 등도 패닝을 가운데로 두는 게 좋다.

- 곡이 진행되는 동안 패닝을 왔다갔다하게 만들 수도 있음 
- 믹서 플러그인 - `Fruity PanOMatic`
![[Pasted image 20251228212241.png]]

- `LFO`에서 `PAN`을 활성화시키고 `AMT, SPD`로 양, 속도를 조절하면 됨

![[Pasted image 20251228212842.png]]
- `Stereo Separation Dial`
	- 왼쪽으로 돌리면 더 넓게(스테레오, 하늘색)
	- 오른쪽으로 돌리면 더 좁게(모노, 보라색)로 들림.

- **드럼의 킥 같은 소리는 모노**로 들리게 오른쪽으로 돌리고
- **화음이나 메인 멜로디는 스테레오**로 구성하도록 왼쪽으로 돌리는 걸 추천.

## 리버브 추가하기
1. 단순히 믹서 채널에 `Fruity Reeverb 2`를 추가하는 방식
	- 특정 악기만의 리버브가 필요할 때 사용할 수 있음
	- 일반적인 프로젝트 구성 방식으로 추천하진 않음
2. **리버브 채널을 별도로 구성함.**
	- 이전에 악기들을 한 채널에 몰아넣은 것과 동일한 방식이다. 리버브만을 위한 채널을 따로 만들라는 것.

> 한 프로젝트에 2개 이상의 리버브를 써도 상관없음. 오히려 권장하는 쪽.

### fruity reveerb 2

![[Pasted image 20251228214218.png]]

- 대부분의 리버브 플러그인은 `wet, dry` 컨트롤을 가진다.
- `dry` 
	- 리버브의 영향을 받지 않은 신호. 원본 개념으로 봐도 무방할 듯.
	- **별도의 채널로 구성했다면 0으로 만들어야 한다.** 별도의 출력 노드가 원본을 재생하고, 리버브 채널에서는 리버브 효과만 재생하게 만드는 작업임.
	- 리버브 채널을 별도로 구성하지 않는 경우 `dry` 값을 살려놔야 함. 안 그러면 `wet` 효과인 울리는 효과만 남기 떄문이다.
- `wet`에 관한 별도의 설명은 없었음..
- `DEC(Decay)` 
	- 값이 작으면 소리가 바로 끝나고
	- 크면 소리가 울리면서 서서히 작아짐


> 딜레이도 리버브처럼 별도의 채널로 구현하고, `Fruity Delay 3`을 사용하며, 딜레이 채널의 플러그인도 `Dry`를 `0`으로 설정해준다.

## EQ
여러 방법이 있음
1. 플러그인 - `Filter - Fruity Parametric EQ 2`
2. 플러그인 목록 아래 그래프를 수정하는 것

강의는 2번 방법으로 진행. 보통 1번으로 작업한다는 듯.

![[Pasted image 20251228220621.png]]
- 아래 다이얼
	- 위 다이얼을 돌려서 각 주역대의 어떤 위치를 수정할지 정할 수 있음
	- 아래 다이얼을 돌리면 좌우 폭을 조절할 수 있음
- 숫자를 조절하면 `주파수(k) 데시벨` 텍스트가 나타남.

> EQ를 만질 때 주의할 점
> - 믹서 악기 하나만을 남겨놓고 작업하면 무엇이 좋은 소리인지 잘 모름
> - **전체 연주 속에서 어떻게 들리는가가 중요하므로 테스트는 채널을 다 켠 상태로 진행**한다.

강의에선 `Lead` 파트에 대해서 `EQ`를 만지고 있는데, 중역대의 경우
- 값을 낮추면 소리가 더 배경에 묻혀서 들리고, 올리면 더 앞으로 나와 뚜렷하게 들리는 경향이 있음

> 일단은 이 정도로만 알면 된다. EQ를 완벽히 다루는 건 오래 걸리므로 많이 만져보는 게 중요함.
