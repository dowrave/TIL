- 특정 목적을 위해 제작된 게 아니기 때문에 매우 다양하게 사용할 수 있음
- FL Studio 20에서 추가되었다고 함

- **검색 간단하게 하기 : `TAGS`**
- 2번째 탭의 하단의 `ALL` 클릭 - 좌측 하단의 `TAGS` 클릭 
	- 목적에 맞는 악기를 검색할 수 있다. `BASS`, `PAD`, `Guitar` 등등.

- **각 노브의 효과는 직접 만져보면서 어떤 느낌으로 동작하는지 하나하나 알아내는 게 가장 나을 듯 - 설명 들어도 그렇게 체감이 안될 때도 있어서 강의만 듣는 건 전반적으로 지루할 수밖에 없음**

- 여러 효과들 팁
	- `ENVELOPES - Volume Env`
		- `Envelopes` - `S(ustain)` + `R(elease)`만땅이면
			- 버튼을 누르고 떼도 소리가 오래 지속됨
			- `S`만 만땅이면 버튼을 누르는 동안에는 소리가 유지됨 (떼면 바로 떨어지지만)
	- `Filter - Env Amt`
		- `Envelopes - Filter Env`의 값들이 얼마나 반영되는지에 대한 영향
	- `Master Filter`
		- 2차 필터 개념. 
		- 하단의 `Low Pass 12`로 된 부분을 바꿔서 어떤 부분을 필터링할지 변경할 수 있음
			- 뒤의 숫자는 필터링하는 강도. `Low Pass 6`이라면 고주파수를 '덜' 필터링한다는 개념
		- 노브값은 얼마나 필터링할 것인지를 결정. 주파수는 위에서 처리하므로 관계 X
		- 참고) 얘가 정상적으로 동작하지 않으면 `FILTER - Cutoff`을 체크해볼 것.
