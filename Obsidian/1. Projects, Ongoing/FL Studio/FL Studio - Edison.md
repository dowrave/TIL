#FLStudio 

- 소리를 녹음하거나, 음원을 편집하는 데 사용함
- `플러그인` 개념임 - 믹서의 마스터에 달 수도, 개별 믹서에 달 수도 있음
## 소리 녹음
- 녹음은 단순하게 `Record` 버튼을 누르면 됨
- **특정 믹서에 `Edison`을 지정해서 그 믹서에 들어오는 소리만 녹음할 수도 있고, 마스터 믹서에 놔서 전체 소리를 녹음할 수도 있음**
- **FL Studio의 해당 채널을 재생시켜서 소리가 나는 상황이면 녹음**된다.

## 영역 지정
- 새로 지정하기 : `Ctrl + 드래그`
- 영역 해제 : 샘플 영역 클릭 후 `윗쪽 화살표`
- 그냥 클릭(드래그)은 이미 지정된 영역에서 추가로 영역을 설정하거나 제거하는 개념
- 영역을 지정한 채 재생시키면 해당 영역만 재생됨

## Scrub
![[Pasted image 20260205015653.png]]
> `Play` 버튼 오른쪽에 있는 아이콘
- 아이콘을 클릭한 채로 좌/우로 움직이면 샘플에서의 위치가 바뀌면서 해당 부분을 재생시킴
- 쓸 일이 많을지는 모르겠음

## 삭제(Delete)
- 영역을 지정하고 `Delete` 키
	- 영역 지정 X 시 전체 삭제
- 삭제된 영역만큼 음원의 길이가 줄어듦
## ★Trim
- **선택 영역만 남기고 나머지를 삭제**
- **`Ctrl + Delete` 키 (=`가위 아이콘 - Trim`)**

- 사용 예시
- 어떤 음원에서 여백을 제거하고 싶을 때, 음파가 있는 부분만 지정하고 `Ctrl + Del`하면 그 부분만 남길 수 있음
- 여러 악기가 섞여 있는 음원에서 특정 파트만 따는 것도 가능함

## Clear
- `Delete`와 달리, **특정 영역의 음파를 제거하되 길이는 그대로 유지**함
- `Shift + Delete`(=가위 아이콘 - Clear)

- 참고) 더 좁은 영역을 크게 보려면 마우스 커서를 파동 영역에 올리고 스크롤

- Delete, Trim, Clear는 `영역 지정 - 우클릭 - Edit`으로 들어가서 쓰는 것도 가능

## 마커 표시
![[Pasted image 20260205021008.png]]
- `Ctrl + 클릭`으로 위치 선택 후, 빨간 동그라미에 위치한 `Add Marker`로 표시 가능
- 사실 **위치 선택 후 `M`으로 마커 지정 및 이름 설정까지 가능**
- 2개 이상의 마커가 있다면 키보드의 방향키로 여러 영역을 왔다갔다 할 수 있음
- 이미 생성된 마커는 우클릭해서 이름을 바꿔줄 수 있음

## Auto-Slicing
- 위의 마커 표시를 자동으로 할 수 있음. **마커 표시의 바로 오른쪽 아이콘.**
- 일시적으로 소리가 나지 않는 영역을 기준으로 자름
- 의도대로 작동하지 않을 수 있음 -> **강사는 이거 먼저 하고 세부 조정하는 식으로 작업**

- 아이콘을 우클릭해서 어떻게 자를지도 설정해줄 수 있다
- `Dull`일수록 덜 자르고 `Sharp`일수록 잘게 자르는 듯. 
- 기본은 `Medium(Alt + S)`

## Normalization
![[Pasted image 20260205022205.png]]
- 샘플의 **피크값을** 찾아 디지털 오디오의 한계치인 **0dB에 도달하도록 파형의 볼륨을 키움**
- 소리의 성질을 조절하지 않고 **게인만을 조절**함
	- `상대적인 다이내믹Relative Dynamic`은 변하지 않음!
	- 그래서 이걸 하더라도 각 음원은 소리가 다를 수 있음
		- 특정 부분의 피크만 높은 음원은 피크에 맞춰서 전체적인 변화량이 크지 않음
		- 편차가 적은, 꽉 찬 음원은 전체적인 변화량이 커짐
- 단축키는 `Ctrl + N`

- 위 **파형의 특정 부분만을 잡아서 정규화하는 것도 가능**함
	- 이 경우 해당 영역의 피크값을 `0dB`까지 올리는 식으로 조절함
- 그래서 `Auto-Slicing`과의 조합도 좋다.

> - 예를 들어서 음원 중에 여러 소리가 나는데, 각 소리가 구분된 하나의 파일로 들어오는 경우가 있음
>- 이 경우, `Auto Slicing`으로 각 소리를 나누고 각각의 영역을 `Normalize`해주면 `0dB` 정도에 해당하는 소리가 나오게 된다. 
>- (물론 이래도 파형마다 다이내믹 레인지가 다르다는 이슈는 해결 안 됨)

## Trim Side Noise
![[Pasted image 20260205023236.png]]
- **음원의 좌우 빈 공간을 바로 없애주는 기능**임
- 모든 경우에 쓸 수는 없음 - 부적절한 상황이 있음

## Gate Noise
- `Trim Side`와 비슷하나, **음원의 길이를 줄이지 않음**
- 위의 `Trim Side Noise` 아이콘을 `우클릭`하면 이 기능이 실행
- 어떤 경우에 쓰는가? : 환경음(노이즈) 같은 거 제거할 때
- 사용 시 주의) 어택이나 디케이도 없어질 수 있음
	- 이를 방지하는 방법이 있음
	1. 음원에서 조용한 부분, 즉 소리의 디케이나 어택감이 남아있는 부분보다 조용한 부분을 찾아서 지정함
	2. 스패너 - `Acquire Noise Threshold` (단축키 `Alt + N`)으로 설정
	3. `Trim Gate Noise` 실행
		- 설정된 부분보다 조용한 부분은 아무 음파도 남지 않음, 음원의 길이도 유지

## Fades and De-click
- `Fade` : 소리가 서서히 커지거나 작아지는 효과
![[Pasted image 20260205024629.png]]
> 위는 `Fade In`, 아래는 `Fade Out`이다.
- 음원의 전체 영역을 잡는 게 아니라, 서서히 변화하기를 원하는 영역만 잡고 설정하면 됨

- `DeClick`
- 우선 예제를 보자. 위 음원에서 양쪽 끝 부분을 Clear하고, 스크롤해서 크게 확대한 모양
![[Pasted image 20260205024945.png]]
**계단처럼 팍 튀어오르기 때문에, 팍 하고 튀는 듯한 소리가 발생**함. 이를 **`Click`** 이라고 한다.

- `DeClick`은 이러한 `Clicking Sound`를 없애주는 역할을 함

- 저렇게 시작하고 끝나는 영역을 잡고, `Fade In / Out 아이콘을 우클릭`하면 됨
![[Pasted image 20260205025148.png]]
