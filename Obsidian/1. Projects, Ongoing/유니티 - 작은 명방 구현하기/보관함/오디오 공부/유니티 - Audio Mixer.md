#오디오 #유니티

- 소리들을 그룹으로 묶어서 한꺼번에 제어하는 믹싱 보드
```
[효과음 1] ──┐
[효과음 2] ──┼──► [SFX Group] ──┐
[효과음 3] ──┘                   │
                                  ├──► [Master]──► 출력
[BGM 1] ────┐                    │
[BGM 2] ────┼──► [BGM Group] ───┘
[BGM 3] ────┘
```

- `Mixer`에서도 효과를 줄 수 있음. 

- 각 믹서 그룹에 연결하는 건 `Audio Source`에서 진행
	- 지정하지 않으면 `Master`에 직접 연결됨

- 사용 예시
	- 설정 화면에서 BGM 그룹만 볼륨 따로 조절
	- 컷씬 진입 시 SFX 그룹만 음소거
	- 실내 진입 시 SFX 그룹만 리버브 적용 

## Snapshot
- 상황에 따라 믹스 설정 전체를 전환하는 기능, 즉 믹서의 프리셋이라고 생각하면 되겠다.
- 만들어 두면 기본 상태 -> 전투 -> 수중 같이 인게임에서 바로바로 전환하는 식으로 사용할 수 있음. 페이드 효과도 자연스럽게 적용 가능.
