1. 오브젝트 풀링 시스템 개선:
   - ObjectPoolManager 클래스를 구현하여 게임 오브젝트의 재사용을 효율적으로 관리하도록 했습니다.
   - 풀 생성, 오브젝트 스폰, 풀 반환, 풀 제거 등의 기능을 구현했습니다.

2. 프로젝타일 관리 최적화:
   - Operator와 Enemy의 원거리 공격에 사용되는 프로젝타일을 오브젝트 풀링으로 관리하도록 했습니다.
   - 각 유닛별로 독립적인 프로젝타일 풀을 가지도록 구현했습니다.

3. 동적 풀 제거 시스템:
   - 유닛이 제거될 때 관련된 프로젝타일 풀도 안전하게 제거되는 시스템을 구현했습니다.
   - MarkPoolForRemoval과 같은 메서드를 통해 풀의 점진적 제거를 가능하게 했습니다.

4. 비동기 풀 제거 처리:
   - RemovePoolWithCallback 메서드를 구현하여 풀 제거 과정을 비동기적으로 처리하고, 완료 시 콜백을 호출하도록 했습니다.
   - MonitorPoolRemoval 코루틴을 사용하여 풀 제거 과정을 모니터링하도록 구현했습니다.

5. Projectile 클래스 개선:
   - MarkForRemoval 메서드를 추가하여 제거 예정 상태를 관리하도록 했습니다.
   - OnDisable 메서드에서 제거 예정 상태를 확인하여 적절히 처리하도록 구현했습니다.

6. 코드 문서화:
   - ObjectPoolManager의 각 메서드에 XML 문서 주석을 추가하여 코드의 가독성과 유지보수성을 향상시켰습니다.
