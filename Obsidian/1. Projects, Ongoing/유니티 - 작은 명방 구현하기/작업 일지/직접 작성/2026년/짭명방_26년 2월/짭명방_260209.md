
1. Sound 조절 패널 
	- 옵션 패널이기도 해서 `ChatGPT`를 통해 `CogWheel01` 이미지 생성하고 넣음
		- Gemini가 이상하게 투명한 배경을 잘 못 만듦. UI에 쓸 이미지는 무조건 `ChatGPT`를 사용하는 중.
	- 이전에는 이미지 사이즈를 조절해서 넣었지만 그냥 1024로 집어넣는다. 최적화가 필요하다면 그 때 하면 될 일임. 지금은 구현에만 집중
	- 일단 구현은 완료. 효과음의 경우 커서를 조절한 후에 소리를 재생시키고 싶어서 `EventTrigger`를 추가하는 방식으로 구현
```cs
void Start() 
{
	// 트리거 부분 구현 - sfxSlider에 EventTrigger는 이미 추가했다고 가정
	EventTrigger trigger = sfxSlider.gameObject.GetComponent<EventTrigger>();
	
	EventTrigger.Entry pointerUpEntry = new EventTrigger.Entry();
	
	// 어떤 트리거 사용할 건지 명시
	pointerUpEntry.eventID = EventTriggerType.PointerUp; 
	
	// 트리거 발동 시 사용할 메서드 정의
	pointerUpEntry.callback.AddListener((data) =>{ OnSFXSliderPointerUp(); }); 
	trigger.triggers.Add(pointerUpEntry);
}

// 이제 OnSFXSliderPointerUp만 구현하면 됨
```

2. BGM 페이드 인 / 페이드 아웃
- 구현 원리
	- `SoundManager`에 `FadeCoroutine`을 추가함
	- `AudioClip.volume`의 값을 코루틴에서 `Mathf.Lerp`로 서서히 조절해나가는 방식. 
	- 코루틴의 중복 실행을 막기 위해 `FadeCoroutine`은 1개만 관리하며, 다른 BGM이 페이드인/아웃되는 상황에 대한 관리도 겸한다.
