```cs
public override void Open()  
{
	base.Open();
	ShowWithLayout();
}   

private void ShowWithLayout()
{
	canvasGroup.alpha = 0f;
	canvasGroup.DOKill();
	canvasGroup.DOFade(1f, fadeInDuration).SetUpdate(true); // 비동기이므로 아래에 코드 작성 주의
}

public void Hide()
{
	canvasGroup.DOKill();
	canvasGroup.DOFade(0f, fadeInDuration)
		.SetUpdate(true)
		.OnComplete(() =>
		{
			gameObject.SetActive(false);
		});
}
```
> `DOFade`부분에 `SetUpdate(true)` 추가 - `unscaledDeltaTime`을 사용함

## 기타 이슈

### 키 바인딩 관련
- `Space, Enter`는 키 변경이 적용되지 않는 이슈가 있다. 이미 중복된 것처럼 처리됨.

#### 로직 자체의 이슈
```cs
// 수정할 요소
foreach (var binding in currentBindings)
{
	if (binding.Value == pressedKey && binding.Key != waitingEntry.Value.action)
	{
		Logger.LogWarning($"{binding.Value}에 중복되는 값이 없음");
		isDuplicate = false;
		return;
	}
}

// 이후 isDuplicate에 따른 처리
```

1. 일단 조건문 후의 처리가 이상했다. `isDuplicate = true`로 한 후에 `break`을 해야 맞았음.
```cs
if (binding.Value == pressedKey && binding.Key != waitingEntry.Value.action)
{
    Logger.LogWarning($"{binding.Value}에 중복되는 값이 없음"); // ← 주석도 반대
    isDuplicate = false; // ← 중복을 찾았는데 false로 설정
    return;              // ← 그냥 리턴해버림
}
```

2. `binding.Value, binding.Key`라는 용어의 의미도 헷갈린다. 딕셔너리의 키랑 단축키의 키라는 의미가 겹치기 떄문.
```cs
foreach (var (action, assignedKey) in currentBindings)
```

#### 최초 문제 처리
- 어제 처리한 내용 중에, 인풋 시스템에 `Enter / Spacebar`으로 처리되는 것 중에는 디폴트로 할당된 요소들이 있다고 했다. 
	- 그거 때문에 내가 설정한 로직이 처리된 게 아니라 유니티 자체 로직이 동작하지 않았을까?

- 이 부분은 "현재 포커스된 버튼"을 해제하는 것만으로 처리할 수 있었음
```cs
// 옵션 섹션에서 키 변경 버튼을 클릭했을 때 동작, 버튼 내의 텍스트를 비우고 키 입력을 기다림(OnAnyKeyPressed)
private void OnBindingButtonClicked(KeyAction action)
{
	BindingButtonEntry entry = buttonEntries.Find(e => e.action == action);

	waitingEntry = entry;
	entry.buttonText.text = "";

	EventSystem.current.SetSelectedGameObject(null); // 버튼 포커스 해제(유니티 자체의 Spacebar, Enter 디폴트 동작 방지)

	// 버튼 입력 시 발동하는 이벤트, CallOnce로 딱 1번 실행 후 자동으로 구독을 해제
	InputSystem.onAnyButtonPress.CallOnce(ctrl => OnAnyKeyPressed(ctrl));
}
```

#### 다른 경우 처리
- `OnBindingButtonClicked`이 동작하고 있을 때 다른 버튼을 클릭한다면 현재 수정 중인 버튼은 기존 상태로 돌아와야 함
