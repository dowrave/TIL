## PopupBase 구현
- 정보를 보여주는 기능만 있는 UI 구현
- 이걸로 프리팹 기본사양을 만들어놓고 추가할 게 있다면 거기서만 추가하면 되는 식으로 작업하는 게 훨씬 빠르긴 했겠다..

### 프리팹을 필드로 저장할 때
```cs
// 이렇게 안 해도 됨 
[SerializeField] private GameObject soundSettingPrefab;

// 이렇게 하면 됨 
[SerializeField] private SoundSettingPopup soundSettingPrefab;
```
> `Instantiate`가 제네릭을 지원하므로 넣은 타입을 그대로 반환해준다. `GameObject` 기반의 `Instantiate`를 할 필요가 없음.
> - 기존에 다 `GameObject` 기반으로 했는데..ㅋ...

- 기타 팝업 기능들을 `PopupBase` 기반으로 수정
