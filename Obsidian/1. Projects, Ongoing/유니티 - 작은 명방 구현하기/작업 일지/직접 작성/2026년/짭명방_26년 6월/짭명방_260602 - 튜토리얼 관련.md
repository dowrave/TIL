
>[!note]
>- 튜토리얼 관련 수정
>	- "조용히 시작한다"는 부분을 없애고, 클리어되지 않았다면 다시 실행됨
>	- 튜토리얼은 "현재"와 "전체"를 언제든지 스킵할 수 있음
>	- 전체 튜토리얼은 옵션 팝업에서 다시 시작할 수 있음
>		- 스테이지 씬에서는 시작할 수 없음

## 튜토리얼 테스트
- 중단되었을 때 해당 부분부터 다시 잘 시작되는지 확인


### 2번째 튜토리얼 재개 관련
- 이게 "조용히 시작한다"랑 겹치는 상황이라 그런 것 같음

- 이런 **실패 조건 체크하느라 튜토리얼에 들이는 품이 너무 커지니까**, 크게 2가지로 압축함

1. 튜토리얼이 조용히 시작하는 부분을 제거함. 즉, 튜토리얼이 진행되고 있는 상태에서 2번째 스테이지에 진입한다면 항상 튜토리얼이 다시 나타남
2. 대신, `TutorialCanvas`에 언제든지 "현재 튜토리얼을 생략"하는 버튼과 "전체 튜토리얼을 생략"하는 버튼을 추가함

- 그래서 실패 조건에 대한 고려도 딱히 안해도 됨. 클리어하지 못했으면 결국 `InProgress`가 되니까. 
```cs
public enum TutorialStatus
{
    NotStarted,
    InProgress,
    // Failed,
    Completed
}
```

- 버튼이 잘 작동하는 걸 확인했음

- 크게 2가지만 더 구현하겠음
1. 각 버튼을 눌렀을 때 `ConfirmationPopup`이 나타나도록 설정
	- `TutorialCanvas`가 있기 때문에 `ConfirmationPopup`을 넣을 수 있다고 생각이 들었음
2. 튜토리얼을 언제든 다시 시작할 수 있는 버튼 하나 추가
	- 옵션 팝업에 "기타" 탭으로 빼서 넣으면 될 듯

#### ConfimrationPopup을 TutorialCanvas에 넣기

```cs
// TutorialCanvas.cs
[SerializeField] private ConfirmationPopup confirmationPopup;

private void HandleSkipCurrentTutorialButtonClicked() 
{ 
	confirmationPopup.Initialize(
		table: LocalizationManager.Tutorial.TableName, 
		key: LocalizationManager.Tutorial.AskSkipCurrentTutorial, 
		isCancelButton: true, 
		onConfirm: SkipCurrentTutorial, 
		onCancel: HideConfirmationPopup);
}

private void HandleSkipAllTutorialButtonClicked() 
{ 
	confirmationPopup.Initialize(
		table: LocalizationManager.Tutorial.TableName, 
		key: LocalizationManager.Tutorial.AskSkipAllTutorial, 
		isCancelButton: true, 
		onConfirm: SkipAllTutorial, 
		onCancel: HideConfirmationPopup);
}


private void SkipCurrentTutorial()
{
	GameManagement.Instance.TutorialManager.OnSkipCurrentTutorialButtonClicked();
	gameObject.SetActive(false);
}

private void SkipAllTutorial()
{
	GameManagement.Instance.TutorialManager.OnSkipAllTutorialButtonClicked();
	gameObject.SetActive(false);
}
```

#### 튜토리얼 재시작 버튼 추가
- `OptionPopup`에 `Others` 탭 추가 후 내부에 버튼 구현
	- 이 버튼은 메인메뉴 씬에서만 동작할 수 있어야 함

- 버튼의 동작은
	1. 튜토리얼 관련 기록 리셋
	2. 튜토리얼 시작
	3. 옵션 팝업을 꺼야 함

```cs
public class OthersSection : MonoBehaviour
{
    [SerializeField] private Button tutorialResetButton; 
    [SerializeField] private OptionPopup optionPopup; // 부모 오브젝트. 간편하게 끄기 위해 사용

    private void Start()
    {
        tutorialResetButton.onClick.AddListener(HandleTutorialResetButtonClicked);
    }

    private void OnEnable()
    {
        // 스테이지 씬에서 튜토리얼 버튼을 비활성화 함. 아니라면 true
        tutorialResetButton.interactable = StageManager.Instance != null ? false : true;

    }

    private void HandleTutorialResetButtonClicked()
    {
        MainMenuPopupManager.Instance.ShowConfirmationPopup(
            table: LocalizationManager.Tutorial.TableName,
            key: LocalizationManager.Tutorial.AskRestartTutorial,
            isCancelButton: true,
            onConfirm: RestartTutorial
            // 취소 버튼 클릭 시 팝업은 자동으로 꺼짐
        );
    }

    private void RestartTutorial()
    {
        // OptionPopup이 닫혀야 함
        optionPopup.gameObject.SetActive(false);
        GameManagement.Instance.TutorialManager.RestartTutorial();
    }

    private void OnDestroy()
    {
        tutorialResetButton.onClick.RemoveAllListeners();
    }
}

// TutorialManager ---------------------------------
public void RestartTutorial()
{
	// 튜토리얼 관련 기록을 리셋함
	GameManagement.Instance.PlayerDataManager.ResetTutorialData();

	// "튜토리얼을 시작합니다" 패널부터 시작
	ConfirmStartTutorial();
}

// PlayerDataManager ---------------------------------
public void ResetTutorialData()
{
	SetIsTutorialRunning(false);
	SetTutorialStatus(0, TutorialStatus.NotStarted);
	SetTutorialStatus(1, TutorialStatus.NotStarted);
	SetTutorialStatus(2, TutorialStatus.NotStarted);
}
```


- 중간에 3번째 튜토리얼이 메인메뉴 씬인데도 시작되는 현상도 있었음. 수정 완료.
