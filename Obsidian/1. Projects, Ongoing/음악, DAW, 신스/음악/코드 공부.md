
## 근음(흰 건반)
### 메이저 코드(3도, 원리는 근음에서 `T T S T`)

1. 샵이 붙지 않는 것들
	- C (도미솔)
	- F (파라도)
	- G (솔시레)

2. 샵이 하나 붙는 것들
	- D (레파#라)
	- E (미라b시)
	- A (라레b미)

3. 샵이 2개 붙는 것
	- B (시레#파#)

> 건반으로 칠 때, 메이저는 반드시 기억해둘 것
> 나머지 코드는 메이저의 응용이다


> 동영상에 나온 건반 이름대로 따라감
> 미-파(E-F) 사이, 시-도(B-C) 사이는 반음으로 연결되는 거만 유의하면 될 듯.
### 마이너 코드
- **메이저 코드에서 3음(가운데 음)을 플랫**

- Cm (도미b솔)
- Dm (레파라)
- Em (미솔시)
- Fm (파라b도)
- Gm (솔시b레)
- Am (라도미)
- Bm (시레파#)

### Diminish 코드
- **메이저 코드에서 근음 빼고 모두 플랫**

- Cdim (도미b파#)
- Ddim (레파라b)
- Edim (미솔시b)
- Fdim(파라b시)
- Gdim(솔시b레b)
- Adim (라도미b)
- Bdim (시레파)

### Augment 코드
- **메이저코드에서 5음을 반음 올림**

- Caug (도미라b)
- Daug (레파#시b)
- Eaug (미라b도)
- Faug (파라레b)
- Gaug (솔시미b)
- Aaug (라레b파)
- Baug (시미b솔)


- 느낌
	- 메이저 : 밝음
	- 마이너 : 슬픔
	- Diminish : 슬픔 이하의 어두움
	- Augment : 증폭되는 느낌(메이저 이후에 오면 더 두드러짐)

## 근음에 뭔 글자가 더 붙음

- 플랫 : 원본 대비 모든 음을 플랫
	- E(미솔#시) -> Eb(미b솔시b)
	- Abaug 
		- A : 라레b미
		- Ab : 라b도미b(모두 반음 내림)
		- Abaug : 라b도미(5음만 반음 올림)

- 샵도 마찬가지로 모든 음의 반음을 올려줌
	- F#m 
		- F : 파라도
		- F# : 파#라#도#
		- F#m : 파#라도#
	- C#dim
		- C : 도미솔
		- C# : 도#파라b
		- C#dim : 도#미솔
