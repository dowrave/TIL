## 3. 화음을 구성하는 음 사이에 음 추가
- 예를 들어 A, C, E로 구성된 화음이라면 B, D 같은 `경과음Passing Note`을 추가하는 것
- 경과음은 **코드에 속하지 않은 음(비화성음)** 이다.

- `강세 경과음Accented Passing Note`
	- 마디의 첫음을 경과음으로 치는 것
	- 해당 마디에 강렬한 효과를 줄 수 있음

- 이외에도 `인접음Neighbor Tone`, `변화음Changing Tone`, `이탈음Escape Tone` 등이 있음.
	- `인접음` : 화성음에서 2도 위 / 아래로 이동했다가 다시 돌아오기
		- 도 -> 레 -> 도
	- `변화음` : 화성음 -> 상행 인접음 -> 하행 인접음 -> 원래 화성음
		- 도 -> 레 -> 시 -> 도
	- `이탈음` : 화성음에서 순차적으로 탈출했다가, 다음 화성음으로 도약해서 이동
		- 도 -> 레(2도 상승) -> 시(3도 하강)
		- 변화음은 원래의 음을 유지하는 성질, 이탈음은 다음 음으로 넘어가기 위한 성질이다.

- 위의 요소를 모르더라도, 아래만 확실히 알아두면 됨
	- **비화음은 긴장**
	- **화음(코드)은 해소**

## 4. 계류음(Suspension), 선행음(Anticipation), 페달음(Pedal Point)
- **비화성음은 다른 코드에서 빌려온 음**이라는 접근

- `계류음Suspension`
	- Am(A-C-E) 에서 G(G-B-D)로 넘어간다면 Am의 마지막 음이 A인 채로 G로 넘어가는 방식
	- **지금 코드의 화음을 다음 코드(에서는 비화음)까지 끄는 개념**

- `선행음Anticipation`
	- Am(A-C-E) 에서 G(G-B-D)로 넘어갈 때, Am의 마디 마지막에 B음을 연주하기 시작해서 다음 마디의 첫음까지 그대로 잇는 방식
	- **다음 코드의 화음을 미리 연주하는 개념**

- `페달음Pedal Tone`
	- 코드 변경에 관계 없이 특정 음을 계속 반복해서 사용하는 것

## 8. 순차적인 진행 / 도약 사이의 균형
- 코드 내에서만 노는 음악은 재미없고, 모든 음이 코드를 벗어난 음악은 혼란스럽다.
	- `순차 진행` : 멜로디에 대한 최소한의 접근 방식
	- `도약` : 노력의 표현
