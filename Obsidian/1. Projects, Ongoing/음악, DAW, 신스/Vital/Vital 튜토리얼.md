## OSC 메뉴 설명

- `Pan` : 이 오실레이터의 패닝
- `Pitch` 
	- 좌측은 세미톤 단위 (-48 ~ 48)
	- 우측은 파인 튜닝 (-100 ~ 100) 
- 하단 `Filter` : 이 `OSC`의 소리를 어디로 보낼 것이냐

![[Pasted image 20260314230024.png]]
- `UNISON`
	- 좌측 : 소리를 내는 갯수. 
	- 우측 : 디튜닝 효과. 
	- `UNISON` 컨트롤은 나중에 `ADVANCED`에서 더 디테일하게 다룰 수 있음
- `PHASE`
	- 상을 바꾸는 기능이지만 이 튜토리얼은 완전 뉴비를 위한 기능이므로 생략

- 2개의 다이얼
	- 좌측 `Spectral Morph` 
		- 다양한 효과가 있고 어떤 느낌인지는 직접 경험해볼 것
	- 우측 `Wave Morph` 
		- 하단 `< --- >` 부분에 어떤 효과를 줄지 고를 수 있다. 이 효과를 주고 다이얼을 돌리면 파형에 변화가 나타남. 
		- 종류마다 효과가 다르므로 직접 들어보면서 어떤 느낌인지 보면 된다. 
	- 튜토리얼에서는 이 다이얼 설정은 따로 하지 않고 넘김

### Sampler
![[Pasted image 20260314231719.png]]
- 노이즈 외에도 기본으로 지원하는 파형이 있고, 드래그 & 드랍으로 갖고 있는 파형을 사용할 수도 있음

- 활용 예시
1. 소리가 너무 디지털처럼 느껴질 때 이런 노이즈 같은 걸 약하게 추가하면 아날로그감 / 따뜻함 등을 추가할 수 있음 
2. 킥 / 스네어 / 퍼커션 소리 같은 일시적인 소리를 추가해서 소리에 임팩트를 줄 수 있음

### Filter
- 필터는 전체 인터페이스의 좌측 하단에 위치함
![[Pasted image 20260314225746.png]]
- `X축` 움직임으로 어떤 주파수 이상을 자를지 고를 수 있음 
	- 어두운 영역이 넓을수록 고음이 많이 잘려서 소리가 부드러워지고
	- 밝은 영역이 넓을수록 소리가 더 거칠어지지만 선명해짐

- 필터도 우측 상단의 `Analog : 12dB` 부분을 클릭하면 다양한 옵션이 있음
- 저 필터 그래프도 잡아서 조절할 수 있음
	- 영역의 디테일한 것까지는 되지 않지만 대략적인 건 가능

- `Blend : 상단의 좌우 스크롤바` 를 만져서 기본 `Low Pass`인 걸 `High Pass`로 바꿀 수도 있다.

- `Mix` : 살짝 내리면 필터링을 완전히 하는 게 아니라 일부를 남길 수 있음
- `Key Tracking` : 입력하는 키에 따라 필터링 그래프가 이동함
	- `Low Pass` 기준 `-100%` : 음이 높아질수록 그래프가 왼쪽으로 이동(고음 컷 증가)
	- `High Pass` 기준 `+100%` : 음이 높아질수록 그래프가 오른쪽으로 이동(더 많은 소리)
	- 어떤 사운드 디자인을 원하느냐에 따라 달라지니까 참고만 하셈

### 우측 하단 메뉴

![[Pasted image 20260314233218.png]]
- `Voices & Polyphony` 
	- 한 번에 몇 개의 음을 동시에 낼 수 있는지를 설정. 
	- 코드를 연주할 때 새로운 음 때문에 기존 음이 잘리는 걸 원치 않을 때 쓴다. 
	- 하지만 많은 게 꼭 좋은 걸 의미하진 않는다.

- `Bend Control`
	- 건반의 피치 휠을 움직일 때 음정이 변화하는 범위를 반음 단위로 설정함
		- `피치 휠` : `FL Studio`에서는 악기 우측 상단의 `Pitch` 노브(`cent`)를 돌릴 때에 적용됨.
	- 기본값은 반음 2개이지만, 12라면 옥타브를 위/아래로 조절할 수 있다.

- `Velocity Tracking`
	- 건반을 치는 강도에 따른 소리 크기 변화

- `Spread`
	- 신스 전체의 스테레오 너비
	- 낮은 값은 모노 사운드, 높은 값은 머리 전체를 감싸는 느낌을 준다.
	- 무조건 넓다고 좋은 건 아님. 응집된 소리가 좋을 때도 있다.

- `Glide(Legato)`
	- 음과 음 사이를 부드럽게 연결하는 효과
	- 항상 켜둘 수도 있고, 레가토 연주 시에만 작동하게 하거나 글라이드의 기울기를 바꿀 수도 있다. 

## Effect 탭
- 각 이펙트는 드래그해서 순서를 바꿀 수 있다. FL Studio의 믹서 플러그인과 비슷한 느낌.
### Chrous
![[Pasted image 20260314234116.png]]
> 크게 언급된 내용은 없음. 제미나이에서 설명을 들고 옴

- 원래 소리에 미세하게 음정 / 시간이 틀어진 복사본을 섞어 소리를 더 넓고 풍성하게 만듦
	- `Voices` : 소리를 몇 개로 복사할 것인가
	- `Frequency` : 음정이 흔들리는 속도. 빠르면 바이브레이션처럼, 느리면 공간감이 생김.
	- `Depth` : 음정이 변하는 정도
	- `Delay` : 원본 / 복사본 소리 간의 시간 차이.
	- `Feedback` : 코러스 효과가 적용된 소리를 다시 입력으로 돌려보낸다. 수치를 높이면 금속성 / 독특한 울림이 생긴다.
	- `Spread` : 복사된 소리가 벌어지는 정도.
### Compressor
![[Pasted image 20260314234303.png]]
- 소리의 크기 차이, 다이내믹 레인지를 줄여서 소리를 단단하고 일정하게 만들어준다.
- `OTT` 이펙트와 유사한 방식으로 작동한다고 함.

- `Mode`
	- `Singleband` : 소리 전체의 볼륨을 하나로 묶어서 조절한다.
	- `Multiband` : 저음, 중음, 고음을 나눠서 압축한다. 특정 대역을 강조하거나 소리를 단단하게 만들 때 사용한다.
- `Threshold`는 그래프에서 직접 만져서 사용함.
	- 윗쪽 그래프 : `High Threshold` - 일반적인 컴프레서의 기능. 너무 튀는 소리를 누름.
	- 아랫쪽 그래프 : `Low Threshold` - 잘 안들리는 소리를 키움. (이게 어색했음)
- `Mix` : 컴프레서가 적용된 소리와 적용되지 않은 소리의 비율을 조절한다.
	- 100%라면 컴프레서 소리만 들리며 값을 줄이면 원래 소리의 타격감 + 컴프레서의 질감이 섞임
- `Low, Band, High`
	- 컴프레싱 후의 최종 게인을 조절한다. 베이스가 부족하면 `Low`를, 선명도를 높이고 싶으면 `High`를 올리는 식.
- `Attack`
	- 소리가 `Threshold`를 넘었을 때 압축을 시작하는 시간. 짧을수록 빠르다.
- `Release`
	- 압축되었던 소리가 얼마나 빨리 원래대로 돌아올지 결정함

### Delay
![[Pasted image 20260314235440.png]]

* 딜레이, 디스토션, EQ, 필터, 리버브는 아는 기능이니까 일단 넘김; 강의에서도 제대로 안 다뤄서..

### Flanger, Phaser
![[Pasted image 20260315000334.png]]

- `Flanger` 
	- 원본 소리에 **아주 짧은 시간 차를 둔 복사본**을 섞어서 만든다.
	- `LFO`로 이를 계속 변화시키면 특정 주파수들이 서로 간섭하며 독특한 소리가 난다.
	- 차갑고 금속적인 느낌이 난다.
	- 일렉 기타를 화려하게 만들거나, 테크노 베이스에 기계 느낌을 더할 때 사용.
- `Phaser`
	- 시간 지연 대신 소리의 위상을 틀어서 **특정 주파수 대역에 노치를 만들고, 이 노치를 위아래로 움직여서 소리를 변하게 만든다.**
	- 특정 구역만 위상을 반전시켜 소리를 상쇄시킨다.
	- 우주적인 느낌 & 플랜저보다 부드럽게 섞인다.
	- 사이키델릭한 패드 사운드나, 펑키한 일렉 기타 / 건반 소리에 입체감을 줄 때 사용
