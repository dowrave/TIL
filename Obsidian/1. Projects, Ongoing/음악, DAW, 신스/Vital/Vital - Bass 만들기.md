## Pluck Bass
- `Pluck` 사운드의 핵심 : **재생 초반에는 하이 필터가 덜 걸리다가, 재생되면서 순간적으로 확 걸리는 것**
- 즉, 필터에 오토메이션을 걸어줘야 한다.

![[Pasted image 20260315225141.png]]
> 그래프는 `Env`여도 되고 `LFO`에 있는 걸 써도 되고 아무거나 써도 됨

- **중요한 건 만든 그래프를 필터의 X축 영역과 연결해줌**
![[Pasted image 20260315225252.png]]
![[Pasted image 20260315225709.png]]

- 예시의 `Env` 그래프에서 가장 바닥이 되는 값이 현재 오토메이션이 걸린 가장 왼쪽 값이 됨
- 그래프의 최고 수치는 **위에 나타난 원 그래프를 클릭 후 위/아래로 드래그하는 식으로 조절할 수 있음** 
- 그게 어느 정도로 증가하는지는 위 그래프의 **청록색 게이지의 길이**로 시각화되어 있음

> 예를 들어 저 길이를 짧게 잡는다면 높은 소리가 항상 많이 잘리므로 `Deep Bass`, 저음 위주의 소리가 많이 날 거고, 저 길이를 길게 잡는다면 고음 소리가 많이 나므로 베이스 외의 목적으로도 사용할 수 있을 것

- 이렇게 연결한 상태에선 엔벨로프 그래프의 속성도 중요하다.
	- 디케이를 길게 잡으면 서스테인 노트로 사용할 수 있고
	- 어택을 살짝 주면 킥감을 낼 수도 있음

- 베이스 자체로는 `thin`, 즉 1개의 `OSC`만으로 구성해도 충분하지만
- 이 예제에서는 `OSC`를 하나 더 추가함

### OSC 2
- `Filter 1`에 연결
- 반음 올림 (`Shift + 스크롤 위`로 바로 가능)
- 유니슨 설정 : 5v, 10%
- 페이즈 설정 : 0, 0%
- 매우 미묘하게 줄 것이므로 Level값을 낮게 줌
![[Pasted image 20260315230844.png]]
> Level이 낮아서 내 귀로는 켰을 때랑 껐을 때랑 솔직히 차이를 못 느끼겠음

- `Subtle`한 두께를 주는 용도임

### Effects
![[Pasted image 20260315232139.png]]

- 디스토션
	- 베이스가 잘 안 들릴 경우, 디스토션을 걸어서 `Bite`감, 즉 날카로움을 더해줄 수 있음
	- 다만 소리가 커져서 잘 들리는 건지, 배음이 추가되어서 잘 들리는 건지 잘 구분이 안 되는 듯 함. 일단 느낌은 전자에 가깝긴 하다
- 컴프레서
	- 다이내믹 레인지를 위해 사용
- 리버브
	- 베이스에 리버브를 걸지 말라는 말이 많지만, 너무 `muddy`하지 않은 선이라면 공간감과 깊이를 주는 데 있어서 베이스에 리버브를 거는 것도 괜찮음
	- 여기선 리버브를 주되 낮은 주파수의 게인값을 대폭 낮추는 식으로 걸어줬음

### 다른 파형 테스트
- 기본적으로 `Saw`에 대해서만 진행했지만
- `Osc 1 : Square`으로 주면 더 일렉트로닉한 느낌이 나고
- `Phase`의 설정을 바꿔서 같은 파형도 변주를 줘서 아예 다른 텍스쳐를 입힐 수 있고
- 무궁무진함

- 어쨌든 `Pluck`에서 가장 중요한 건 **그래프를 `Filter`의 기준선에 연결해주는 것**

## Grawl Bass
- `OSC 1`
	- 파형 : `Quad Saw`
	- 피치 : `-12`
	- `Frequency Morph Type` : `Smear`, 게이지 아주 약간 올림

- `OSC 2`
	- `Frequency Morph Type` : `Smear`, 게이지 아주 살짝 변경
	- `Unison` : `10v, 8%`
	- `Level` 낮춤

- `Effects - Filter`
	- `Resonance` 값 매우 낮춤
	- `Cutoff`도 가장 오른쪽으로 붙임 : 즉 거의 필터링되지 않게

- `Filter 1`
	- 위의 `Effects - Filter`와 비슷하게 설정


![[Pasted image 20260315235526.png]]
- `Filter 2`
	- `Osc 2`에 연결해서 사용
	- `Blend`을 오른쪽 끝으로 땡겨서 하이패스 필터처럼 사용

- `Random 1`을 `OSC 2 - Spectral Morph`에 연결, 상대적으로 변화값이 크게
- `Random 2`을 `OSC 1 - Spectral Morph`에 연결, 상대적으로 변화값이 작게

- 즉 OSC1, 2의 `Frequency Morph` 값이 변하되 상대적인 고음을 담당하는 OSC 2의 변화가 조금 더 많고 저음인 OSC 1의 변화가 적은 느낌이 되겠다
