## 260128

### 이슈 수정 - PathIndicator
- `PathIndicator` : 지정된 경로가 아니라 이상하게 이동하는 현상
	- `Enemy`와 똑같이 `PathNavigator`를 사용하는데 얘만 이슈가 발생하고 있음
	- 경로 데이터가 이상한 것도 아니고(Enemy와 공유하고 있는데 Enemy는 잘 움직임)
	- 경로가 업데이트되는 것도 아닌데.. 흠
	- 발견) 초기화 시점의 `currentDestination`이 `0, 0, 0`이라 그런 것 같음
	- **`currentDestination` 기반으로 동작**하도록 수정함

- 애초에 `Enemy`랑 `PathIndicator`의 로직이 (다르긴 해야 하지만) 다른 상태로 있던 것도 문제임
- 지금까지 컨트롤러들을 구현해오던 규칙이 있고, 이동 로직도 컨테이너에서 구분해야 하기 때문에 이참에 아예 싹 정리하겠음

### 이동, 경로 컨트롤러 분리
- `MovementController, PathController`로 사용.
- 구현은 거의 다 했고 테스트 진행 중, 코드는 여기에 나중에 정리
- 현재 이슈 : `PathIndicator`에 노드들이 제대로 전달되지 않고 있음
	- 지금까지 잘 되다가 갑자기 노드가 0개로 나오는 현상이 있다. 
	- 인스펙터에서도 노드 갯수 정상적으로 나오고..
