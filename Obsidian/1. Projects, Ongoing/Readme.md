>[!wip]
>1. 효과음 구현
>	- 생성 / 탐색 : `ElevenLabs` / `FreeSound` / `PixaBay`
>	- 수정 : `FL Studio`
>	- AI 딸깍이라고 쉬운 건 아니다! 를 실감하는 중
>	- 이외 기타 등등은 새로 만들거나 애셋, 사이트 등을 뒤져보자
>2. 발견한 버그
>	- `AreaHasteHeal` 스킬 도중 퇴각했을 때 `AreaHasteHealController`가 사라지지 않고, 다시 배치할 때 이펙트도 그대로 같이 나오고 있음

>[!note]
> 소리 크기 설정 관련 : **다른 매체(게임, 인방 등등)와 사운드 크기를 비교해가면서 기준을 잡아야 할 것 같다.** 
> - 평소에 헤드폰으로 테스트하고 있었는데, 사운드 테스트 때문에 다른 매체에서 나는 소리를 아예 배제하면서 테스트했더니 오히려 비교를 못 하게 되었고 그래서 내 프로젝트의 사운드만 확연히 작은 이슈가 발생하고 있음. 
> - `260212` 기준, **지금은 확실하게 다른 매체보다 소리가 작음.** 얼마로 설정했는지는 파일 이름에 명시해뒀다.
> 아래 표에서 BGM의 LUFS 값 기준을 `-20`으로 잡고 테스트해야 할 듯. 

|    유형     | **LUFS** |                LUFS 판단 기준                 |
| :-------: | :------: | :---------------------------------------: |
| 일반 타격, 마법 | **-16**  | Momentary Max(400ms) / Short Term Max(3s) |
|  UI/시스템   | **-20**  | Momentary Max(400ms) / Short Term Max(3s) |
|    BGM    | **-24**  |                Integrated                 |
